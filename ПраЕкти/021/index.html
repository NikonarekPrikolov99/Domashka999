HATOR

HATOR ROCKFALL TKL

Hator Rockfall TKL – перша на ринку ігрова клавіатура в компактному форм-факторі, в якій встановлено оптичні перемикачі. Надійність конструкції, моментальна реакція, вбудована пам’ять – ця клавіатура просто створена для LAN-змагань. Тепер ти – найшвидший гравець на турнірі!

ХАРАКТЕРИСТИКИ

Кількість клавіш: 87
Тип піключення: USB
Частота опитування: 1000 Гц
Підсвітка: RGB на 16.8 млн. кольорів
Тип перемикачів: Kailh Optical Switches (Black)
Ресурс перемикачів: 80 млн. циклів
Сила натискання: 45 г «+/-»10% для Kailh Black та 55 г «+/-»10% для Kailh Blue.
Затримка до спрацьовування: 0.2 мс
Хід до спрацьовування: 1.8 «+/-» 0.6 мм.
Загальний хід: 3.6 «+/-» 0.6 мм.
Засіб нанесення символів: Лазерне гравірування.
Підтримка операційних систем: Windows XP/7/10, Linux та Mac OS.
Комплектація: Клавіатура Hator RockFall TKL, Пристрій для заміни перемикачів, Інструкція
Додаткові особливості: Металева пластина всередині надає високу міцність
Додаткові особливості: Можливість зміни перемикачів на інший тип або їх комбінування між собою
